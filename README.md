# dotfiles

## Usage
```bash
git clone [ssh/http]
cd dotfiles
bash ./bootstrap.sh
```
## Switching to zsh 
install
```
sudo apt update
sudo apt install zsh -y
```

swtich to zsh
```
chsh -s $(which zsh)
```

install oh-my-zsh
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
